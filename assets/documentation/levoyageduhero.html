Le Voyage du Héros

Application de structuration narrative

Description du Projet

Le Voyage du Héros est une application desktop Windows permettant aux auteurs de structurer leurs récits narratifs selon les 12 étapes du Voyage du Héros. L'application offre des outils visuels pour organiser les idées, planifier les intrigues et maintenir une cohérence narrative.

Fonctionnalités

Création de projet

  • Création de nouveaux projets avec titre et description
  • Gestion de plusieurs frises par projet
  • Stockage des données en base SQLite

Gestion des notes

  • Notes associées aux étapes du héros
  • Éditeur Markdown/RichTextBox
  • Système de tags personnalisables
  • Positionnement sur la frise temporelle

Visualisation et analyse

  • Frise interactive pour la vue d'ensemble
  • Statistiques narratifs avec OxyPlot
  • Navigation rapide avec raccourcis clavier
  • Zoom temporel pour les scènes précises

Exportation et collaboration

  • Export PDF du plan narratif
  • Import/export JSON pour le partage
  • Mode lecture seule pour les bêta-lecteurs

Technologies Utilisées
