Client Lourd - Le Voyage du Héros

Application de gestion de projets narratifs

Description du Projet

Le Voyage du Héros est une application web permettant aux utilisateurs de gérer et de suivre leurs projets narratifs en suivant la structure du "Voyage du Héros" de Joseph Campbell. L'application aide les créateurs à organiser leurs idées, à suivre leur progression et à collaborer avec d'autres auteurs.

Modèle Conceptuel de Données

Structure de la Base de Données

Le MCD est organisé autour de plusieurs entités principales :

  • UTILISATEUR : Gère les comptes utilisateurs et leurs préférences
  • PROJET : Représente les projets narratifs créés par les utilisateurs
  • ETAPE : Définit les différentes étapes du voyage du héros
  • NOTE : Permet aux utilisateurs de prendre des notes sur leurs projets
  • RESSOURCE : Stocke les ressources associées aux étapes
  • COMMENTAIRE : Facilite la collaboration entre utilisateurs
