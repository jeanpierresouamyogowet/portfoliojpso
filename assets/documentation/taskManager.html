Task Manager

Application de gestion de tâches

Description du Projet

Task Manager est une application de bureau développée en C# avec Windows Forms permettant de gérer efficacement vos tâches quotidiennes. L'application utilise SQLite comme base de données et Dapper comme ORM pour une gestion optimale des données.

Fonctionnalités

Gestion des tâches (CRUD)

  • Création de nouvelles tâches avec nom, dates et priorités
  • Affichage de toutes les tâches enregistrées
  • Modification du statut et de la priorité
  • Suppression des tâches

Base de données SQLite

  • Création automatique de la base de données
  • Table Todo avec tous les attributs nécessaires
  • Utilisation de Dapper pour les requêtes SQL

Interface Windows Forms

  • Interface utilisateur intuitive
  • Affichage des tâches dans un DataGridView
  • Formulaires pour l'ajout et la modification

Modèle de Base de Données

Table Todo

Attribut Type Description
Id INTEGER Clé primaire, auto-incrémenté
Nom TEXT Nom de la tâche (obligatoire)
Date_debut DATETIME Date de début (obligatoire)
Date_fin DATETIME Date de fin prévue (facultatif)
Statut TEXT Statut de la tâche
Priorite INTEGER Niveau de priorité

Compétences Développées

Développement d'applications Windows avec C# 7.3 et Windows Forms.

Gestion de base de données SQLite et écriture de requêtes SQL.

Utilisation de Dapper pour le mapping objet-relationnel.

Organisation du code en couches et gestion des dépendances.

Création d'interfaces Windows Forms et liaison de données.
